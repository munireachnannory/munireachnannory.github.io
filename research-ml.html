ML for Inverse Property Extraction

Given only a force–displacement curve from a nanoindentation experiment, can a machine learning model accurately recover the underlying material properties — elasticity, hardness, and plasticity — without solving the costly inverse FEM problem? This work says yes.

The inverse problem

Forward nanoindentation FEM is straightforward: given material properties (E, H, σ_y, n), simulate the indentation and get a load–displacement (P–h) curve. The inverse problem — going from a measured P–h curve back to material properties — is ill-posed, nonlinear, and computationally expensive when solved by iterative FEM.

Traditional approaches iterate FEM runs to minimize the difference between simulated and measured curves. This can take hours per sample. A trained ML surrogate can perform the same inverse prediction in milliseconds, enabling high-throughput materials characterization.

Research question

Can a data-driven surrogate model trained on FEM-generated P–h curves reliably extract E, H, and σ_y for unseen experimental measurements — including thin films where substrate effects must be accounted for?

Model architecture

01

Feature extraction from P–h curves

The raw load–displacement curve is reduced to a compact feature vector: Oliver-Pharr stiffness, unloading slope, maximum load, residual depth, area under curve, and key inflection points. These physics-informed features encode the material response compactly.

02

Deep Kernel Learning (DKL)

A neural network learns a deep feature representation, which feeds into a Gaussian Process kernel. This hybrid architecture inherits the flexibility of neural networks and the uncertainty quantification of Gaussian Processes — giving not just predictions but confidence intervals on E, H, and σ_y.

03

Active learning for data efficiency

Rather than uniformly sampling the parameter space, an active learning loop selects the next simulation to run based on model uncertainty. This dramatically reduces the number of FEM simulations needed to achieve a target prediction accuracy.

04

Uncertainty-aware predictions

For each experimental input, the model outputs a mean prediction and a confidence interval for each property. Samples with high uncertainty are flagged for additional experimental validation — enabling a human-in-the-loop quality control system.

Training & validation strategy

The model is trained entirely on FEM-generated P–h curves, spanning a broad range of E (50–400 GPa), yield stress σ_y (0.1–5 GPa), and hardening exponents. It is then validated on:

Copper (Cu) wire — bulk Molybdenum (Mo) — bulk Tungsten (W) — bulk Cu thin films on Si High-entropy alloys (future)

Validation metrics include mean absolute percentage error (MAPE) on E and H, and comparison against reference values from independent characterization (e.g., EBSD, XRD, tensile testing).
